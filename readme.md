# SafeQuery: Ecommerce Sales Analytics Agent

## Project Overview
SafeQuery is a structured agentic AI system that converts natural language ecommerce analytics queries into safe SQL and executes them on a PostgreSQL database.

The system demonstrates:
- Planner → Generator → Validator → Executor architecture
- Safe SQL execution
- Multi-table joins
- Automated visualization using Matplotlib

---

## Domain
Database Agent – Ecommerce Sales Analytics

Database Tables:
- customers
- products
- orders
- order_items
